# 工具类 (utils)

## 文件列表

| 类 | 功能 |
|----|------|
| `ActivityUtils` | Activity 跳转工具 |
| `CrashLogUtil` | 崩溃日志收集（SharedPreferences） |
| `DeviceUnit` | 设备信息、剪贴板、分享 |
| `DownloadUtil` | APK 下载安装（用于应用内更新） |
| `FileUtil` | 文件操作、权限检查 |
| `LogUnit` | 日志打印（统一标签） |
| `NetUnit` | 网络相关（URL 解析、连通性检查） |
| `PackageUtil` | 版本号读取 |
| `TextUnit` | 文本处理（拼接、截断） |
| `ToastUnit` | Toast 封装 |
| `WebUnit` | 浏览器跳转 |

## thread/ — 线程工具

| 类 | 功能 |
|----|------|
| `AppLogTask` | 后台日志写入任务 |
| `ThreadPoolUtil` | 线程池管理 |
